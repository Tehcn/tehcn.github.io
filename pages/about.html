About this website

This website is built using a custom solution. I use a file system based router because my old solution (a JSON metadata file) drove me insane.

All of the links in this website actually call a JavaScript function called loadArticle which takes the URL of the page as a parameter. A fetch request is performed and the websites main article tag's contents are replaced with the new page's contents (making this an SPA).
